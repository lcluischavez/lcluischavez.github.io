Projects

Below are some projects I helped on and some that I built myself. In order they show the progress of my abilities and skills.

Nutshell (group project)

Nutshell was a group project around the halfway mark at my time at Nashville Software School. It was a 5 day sprint between 3 other classmates and myself. The goal of the app was to code a mock app of Facebook where users would be able to add new friends to their friend lists, which would alow them to then message those users. Users of the web app would also be able to have full crud on components like events and blog type posts. Another feature would alow users to have crud functionality on a Todo List that only they could see individually after logging in. I was in charge of CRUD functionality on the Events, Todo Items, and Post components. Technologies used in my application include React, JSX and CSS.

EPLP

This website is for individuals with a mass collection of album and vinyls . The application gives the user the ability to catalog and organize the collection. The user has the can Add, Delete and Edit albums. They can also create artists and assign albums to those artists. Once the user has created an album and assigned an artist to it they can also Add and Delete songs to that specific album as well. The application also lets users create playlists or "Mixtapes" with music they already have. Technologies used in my application include React, JSX and CSS.

Roads

Website was intended for car enthusiasts and it's a great way for people interested to find their first events. This app lets users post and RSVP to events so users can find events of the type they like. This app also lets users add their car builds to share with other users. Users also have the ability to post parts for other users to buy. After getting feedback from the community I'm currently working on adding some extra features. Technologies used in my application include C#, .Net, HTML and CSS.
